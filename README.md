### Quick Start

#### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

#### 2. Run the Application

```bash

```

- **Main API**: http://localhost:8000
- **Interactive Docs**: http://localhost:8000/docs

## 🔷 .NET WebAPI

A modern ASP.NET Core WebAPI implementation using .NET 9.0.

### Features

- ✅ **RESTful API** - Modern .NET WebAPI with controllers
- ✅ **Swagger Documentation** - Built-in OpenAPI support
- ✅ **Minimal API** - Clean and efficient endpoint definitions
- ✅ **Production Ready** - Optimized for performance and scalability
- ✅ **Cross Platform** - Runs on Windows, macOS, and Linux

### Quick Start

#### 1. Prerequisites

Ensure you have .NET 9.0 SDK installed:

```bash
dotnet --version  # Should show 9.0.x
```

#### 2. Run the Application

```bash
cd WebApiProject
dotnet run
```

The API will be available at:
- **Main API**: https://localhost:7298 (HTTPS) or http://localhost:5000 (HTTP)
- **Swagger UI**: https://localhost:7298/swagger

#### 3. Test the Endpoints

```bash
# Weather forecast endpoint (default)
curl https://localhost:7298/weatherforecast

# Or with HTTP
curl http://localhost:5000/weatherforecast
```
